Using the url tag

  • hard-coded (not DRY)
  • Using DTL: {% url 'route:first-view' %} is the command: "{'% url 'route:first-view' %'}" Which refernce to: (url 'route:first-view')
  • url 'route:second-view'
  • {% url 'gview:cats' %} (url 'gview:cats') from gview/urls.py
  • {% url 'gview:cat' 42 %} (url 'gview:cat' 42) from gview/urls.py
  • {% url 'nsroute:second-view' %} (url 'nsroute:second-view') from dj4e-samples/urls.py
